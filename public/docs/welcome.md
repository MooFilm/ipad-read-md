# เริ่มต้นใช้งาน ReadMD

เว็บแอพนี้ออกแบบมาเพื่อให้อ่านไฟล์ Markdown บน iPad ได้สบายขึ้น โดยเฉพาะเอกสารยาว ๆ เช่น lecture note, project log หรือ thesis draft

## ฟีเจอร์หลัก

- โหลดไฟล์ `.md` จากโฟลเดอร์ `public/docs`
- จำตำแหน่งการอ่านล่าสุดของแต่ละไฟล์
- มีปุ่ม bookmark สำหรับมาร์กจุดที่อยากกลับมาอ่าน
- ใช้งานบน Safari ได้ และเตรียมต่อยอดเป็น PWA ได้

## วิธีเพิ่มไฟล์ใหม่

1. วางไฟล์ใหม่ในโฟลเดอร์ `public/docs`
2. เพิ่มข้อมูลไฟล์ในตัวแปร `docs` ภายใน `src/App.jsx`
3. รันโปรเจกต์แล้วเลือกเอกสารจากแถบด้านซ้าย

## แนวคิดต่อยอด

- เพิ่มระบบค้นหาในเอกสาร
- แยกหมวดวิชา/หัวข้อ
- ซิงก์ตำแหน่งอ่านด้วย GitHub Gist, Supabase หรือ Firebase
- เพิ่ม dark mode

## ทดลองเลื่อนลงมา

ยิ่งเลื่อน ระบบจะจำเปอร์เซ็นต์ที่อ่านไปเรื่อย ๆ หากรีเฟรชหน้าแล้วเปิดกลับมาใหม่ ระบบจะพยายามเลื่อนไปที่ตำแหน่งเดิมให้อัตโนมัติ

### หมายเหตุ

หากเปิดคนละ browser หรือคนละเครื่อง ค่าที่จำไว้ใน `localStorage` จะไม่ซิงก์กันเอง นี่เป็นเวอร์ชันเริ่มต้นที่เหมาะกับการทำ prototype ก่อน
